## Project Overview
This repository contains a **Power BI dashboard** that analyzes a sales dataset to reveal revenue patterns, product performance, regional trends, and business insights.  
Using this interactive Power BI report, stakeholders can explore key sales metrics and improve decision-making.

---

## Objectives
- Visualize **total sales trends** over time
- Identify **top selling products**
- Compare **regional sales performance**
- Evaluate **profit & revenue behavior**
- Provide interactive **insights using slicers and filters**
